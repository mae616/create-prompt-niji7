# 情報密度・視線誘導・空間設計

## メタデータ
- priority: 5
- conflicts: []

## ルール

### 情報密度と主従
- 画面内を同じ密度で描かない
- 主役（特に顔や表情）は比較的整理する
- 衣服や背景は、省略・単純化・色の塊として扱って構わない
- 人物と背景の境界は、色や筆致が重なる表現を許容する

### テーマ解釈と視線誘導
- 主役は必ず一つに寄せる
- 視線が最初に向かう場所を明確に設計する
- 主役周辺は、形・明度・色相の読み取りやすさを優先する
- 背景や周辺要素ほど、にじみ・情報の粗さを強めて構わない

### 視線の流れの設計
- 最初に目が行く場所（第一視点）
- 次に自然に目が移る場所（第二視点）
- 最後に全体として眺める領域（第三視点）
- この順序を前提として、プロンプト内の情報密度・描写の強さ・コントラストを配分する

### 着色と視線誘導の関係
- にじみ・重ね塗り・荒れた筆致は視線を散らす要素として作用する前提で扱う
- 見せたい部分にはにじみを「コントロールされた形」で使う
- 見せなくてよい部分にはにじみ・色の重なり・形の崩れを積極的に使ってよい
- 結果として視線が自然に主役へ集まり、そこから周囲へ流れる構成を目指す

### 空間設計
- 空間タイプを1つ指定（例: 室内奥行き型 / 平面色面型 / 斜構図通路型 / 開放高所型）
- 奥行き構造を明示（前景 / 中景 / 背景）
- 主役との距離（バストアップ / 半身 / 全身）
- 視線誘導構造（対角線 / 放射 / 面の切り替え）
- 背景の内容（短文で1行）
- 背景の密度は主役より必ず低密度

### 空間の制約
- 主役を圧迫しない
- 主役と同密度禁止
- 空間は色面で成立

### 最優先原則
- 「絵画的であること」よりも「何を見せたい絵かが一瞬で伝わること」を優先する

## キーワード
- 視線誘導: focal point, visual hierarchy, clear subject, clear density contrast between subject and simplified background
- 密度差: detailed focal area, simplified background, painterly periphery
- 境界処理: soft edge transition, blending boundary between subject and background
- 空間: interior depth space, open outdoor space, diagonal corridor composition

## 例
- 良い例: "highly detailed face and expression as focal point, clothing simplified into color masses, background dissolving into soft color washes, clear density contrast between subject and simplified background"
- 悪い例: "everything equally detailed"（均一な情報密度は禁止）
